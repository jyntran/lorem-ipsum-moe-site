API Documentation

http://moe.jyntran.ca/api/get

Limitation

The API is free to use. The only limitation at this time is the request rate, which is set to 120 requests a minute (~2 requests a second). This may change depending on demand.

Examples

    /get?p=2&lorem=true
    Returns two paragraphs beginning with Lorem Ipsum. 
    /get?s=4&w=6
    Returns four sentences with six words each. 
    /get?p=3&format=text
    Returns three paragraphs as plaintext.

Queries

    p
  • Type: integer
  • Description: Number of paragraphs.
  • Default: 3
    s
  • Type: integer
  • Description: Number of sentences.
  • Default: 6
    w
  • Type: integer
  • Description: Number of words.
  • Default: 8
    lorem
  • Type: boolean
  • Description: Start with Lorem Ipsum.
  • Default: false
    nsfw
  • Type: boolean
    • Description: Include NSFW words.
  • Default: false
    format
  • Type: string
  • Description: Format the response. Options: json, text, html
  • Default: json
