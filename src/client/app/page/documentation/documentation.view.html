API Documentation

http://moe.jyntran.ca/api/get

Examples

    /get?p=2
    Returns two paragraphs. 
    /get?s=4&w=6
    Returns four sentences with six words each.

Queries

    p
  • Type: integer
  • Description: Number of paragraphs.
  • Default: 3
    s
  • Type: integer
  • Description: Number of sentences.
  • Default: 6
    w
  • Type: integer
  • Description: Number of words.
  • Default: 8
    lorem
  • Type: boolean
  • Description: Start with Lorem Ipsum.
  • Default: false
    nsfw
  • Type: boolean
    • Description: Include NSFW words.
  • Default: false
    format
  • Type: string
  • Description: Format the response.
    • json returns an array of paragraphs.
    • text returns plaintext.
    • html returns text wrapped in paragraph tags.
  • Default: json
